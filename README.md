# ecommerce-user-behavior-analysis
Analyze 450K+ e-commerce user behavior records. Identify key drop-off points in conversion funnel, benchmark mobile vs desktop performance, and propose data-driven strategies to boost conversion rate.  （基于 45 万条电商用户行为数据，定位转化漏斗核心流失节点，对比移动端与电脑端转化差异，提出数据驱动的优化策略，助力提升购买转化率。）
